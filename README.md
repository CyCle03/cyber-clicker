Cyber Clicker is an incremental/idle clicker game with a cyberpunk aesthetic. Your goal is to hack the system, accumulate bits, purchase upgrades, and eventually gain root access levels to unlock permanent bonuses.

## âœ¨ Features

### Core Gameplay
- **Manual Clicking**: Click the HACK_SYSTEM button to generate bits
- **Automatic Production**: Purchase upgrades to generate bits per second (GPS)
- **Prestige System**: Reboot the system to gain Root Access levels and permanent multipliers
- **Skill Tree**: Unlock skills using skill points earned from reboots
- **Achievements**: Complete various challenges to unlock achievements

### Game Systems
- **Upgrades**: 13 different upgrades from Mech Switch to Reality Bender
- **Black Market**: Purchase temporary boosts and permanent upgrades using Cryptos
- **Firewalls**: Random events that reduce GPS until cleared
- **Glitches**: Random events that reward Cryptos when clicked
- **Data Breach**: Mini-game to earn bonus bits
- **Offline Progress**: Continue earning bits while offline (up to 7 days)

### Statistics & Tracking
- Total clicks
- Total bits earned
- Play time
- Reboot count
- Firewalls encountered and cleared

## ğŸ¯ How to Play

1. **Start Hacking**: Click the HACK_SYSTEM button to generate bits
2. **Buy Upgrades**: Use bits to purchase upgrades in the SHOP tab
3. **Earn Cryptos**: Complete achievements or click glitches to earn Cryptos
4. **Black Market**: Spend Cryptos on temporary boosts or permanent upgrades
5. **Reboot System**: When you have enough lifetime bits, reboot to gain Root Access levels
6. **Unlock Skills**: Use skill points from reboots to unlock skills in the Skill Tree
7. **Progress**: Continue upgrading and rebooting to reach higher levels

## ğŸ“‹ Game Mechanics

### Bits
- Primary currency
- Generated by clicking or through GPS (bits per second)
- Used to purchase upgrades

### Cryptos
- Secondary currency
- Earned from achievements and glitches
- Used in the Black Market

### Root Access Levels
- Gained by rebooting the system
- Each level provides +10% GPS bonus
- Requires exponentially increasing lifetime bits
- Grants skill points for the Skill Tree

### Skills
- **Click Efficiency**: Increases click power by 50% per level
- **GPS Overclock**: Increases GPS by 10% per level
- **Firewall Bypass**: Reduces firewall penalty by 10% per level
- **Lucky Hacker**: Increases glitch spawn rate
- **Crypto Magnet**: Increases crypto drops
- **Offline Optimizer**: Increases offline earnings by 5% per level
- **Prestige Master**: Reduces reboot requirements by 10% per level
